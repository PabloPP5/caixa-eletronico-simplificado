# Caixa Eletronico em C

Mini-projeto desenvolvido em C com o objetivo de praticar lógica de programação, estruturas condicionais, laços de repeticao e manipulacao de variaveis.

O programa simula o funcionamento basico de um caixa eletronico, permitindo realizar operacoes de deposito, saque e consulta de saldo, alem de gerar um relatorio final.

## Funcionalidades

- Registro de multiplas operacoes
- Deposito de valores
- Saque com verificacao de saldo
- Consulta de saldo
- Relatorio final com:
  - Total depositado
  - Total sacado
  - Saldo final
  - Maior deposito
  - Maior saque

## Objetivo do projeto

Treinar conceitos basicos da linguagem C, principalmente:
- Estruturas condicionais (if / else)
- Estruturas de repeticao (for)
- Operadores ternarios
- Controle de fluxo
- Logica de programacao

## Como executar

Compile e execute o arquivo principal:

```bash
gcc caixa_eletronico.c -o caixa_eletronico
./caixa_eletronico
```
